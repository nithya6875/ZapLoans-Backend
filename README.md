# Zaploans-Backend

## Getting Started

Create a `.env` file in the root directory and add the following environment variables:

```env
DATABASE_URL=<DATABASE_URL>
PORT=<PORT>
```

To install dependencies:

```bash
bun install
```

Generate Drizzle client:

```bash
bunx drizzle-kit generate
```

To run:

```bash
bun run start
```
